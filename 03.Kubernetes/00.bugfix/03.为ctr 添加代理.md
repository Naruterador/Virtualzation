1️⃣ 创建 containerd 的 systemd drop-in 目录
`mkdir -p /etc/systemd/system/containerd.service.d`

2️⃣ 写代理配置文件
```vim
vim /etc/systemd/system/containerd.service.d/proxy.conf

[Service]
Environment="HTTP_PROXY=http://127.0.0.1:7890"
Environment="HTTPS_PROXY=http://127.0.0.1:7890"
Environment="NO_PROXY=localhost,127.0.0.1,::1,10.0.0.0/8,192.168.0.0/16"
```

3️⃣ 重载并重启 containerd
```shell
systemctl daemon-reexec
systemctl daemon-reload
systemctl restart containerd
```

4️⃣ 验证代理是否真的生效
`systemctl show containerd --property=Environment`
